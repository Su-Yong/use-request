# Cache
`Cache` must implement all 5 methods below.
* `get(id: string): Data`
* `set(id: string, data: Data)`
* `delete(id: string)`
* `has(id: string): boolean`
* `clear()`

JavaScript's `Map` can be used as `Cache` as it satisfies all five conditions above.
